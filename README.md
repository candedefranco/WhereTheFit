# WhereTheFit
Saw an outfit you loved and don't know where to get it? WhereTheFit is for you.

WhereTheFit is a fashion-focused social network built for the Argentine market. Users post photos of pieces they're looking for and the community responds with store links, showroom addresses, Instagram profiles, and anything else that helps track down the item.

### How it works

Post a photo of the piece you're looking for — a screenshot, a street photo, anything.
The community comments with leads: store links, Instagram profiles, showroom addresses.
Mark your post as resolved once you find what you were looking for, and share where you got it to help others.


### Features

- Upload up to 3 photos per post with an Instagram-style carousel
- AI-powered tag suggestions (Google Gemini) based on image analysis
- Threaded comments with nested replies
- Resolution system — mark your search as resolved and share where you found it
- Categories and tags to organize posts
- User profiles with active and resolved search history
- Follow system to connect with other users
- Pinterest-style masonry feed, fully responsive


### Tech Stack

Backend --> Python + Flask
Database --> PostgreSQL
ORM --> SQLAlchemy
Auth --> JWTImage 
storage --> AWS S3
AI --> Google Gemini
Frontend --> React + TypeScript + Vite
Layout --> react-masonry-css
